# Product Overview

GPT App内のチャットインターフェースから天気予報情報を取得すると同時に、x402プロトコルを通じてステーブルコイン支払いが自動的に行われるサンプルアプリケーション。

## Core Capabilities

1. **天気予報API提供**: x402バックエンドサーバーを通じた天気予報データの配信
2. **自動マイクロペイメント**: x402プロトコルによるAPI呼び出し時のステーブルコイン自動決済
3. **GPT App連携**: MCPサーバーを介したGPT Appチャットインターフェースからのシームレスなアクセス

## Target Use Cases

- GPT App内から自然言語で天気予報を問い合わせ、裏側で自動的にステーブルコイン決済が発生するデモンストレーション
- x402プロトコルを活用したWeb3ペイメント統合のリファレンス実装
- MonorepoとCloudflare Workersを使ったマイクロサービスアーキテクチャの学習素材

## Value Proposition

- x402プロトコル（HTTP 402 Payment Required）を実際のユースケースで体験できるサンプル
- GPT AppとWeb3決済の統合パターンを示す実用的なリファレンス
- VibeKanbanとGit Worktreeを活用した並列AI駆動開発のショーケース
